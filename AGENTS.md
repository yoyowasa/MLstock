## Workspace Rules
- この workspace `C:\BOT\MLStock` は、MLStock の週次 US equity pipeline と gap 戦略の実装・検証用として扱う。
- 既存の `C:\BOT\MLStock\AGENT.md` の言語・出力方針も併せて守る。
- 他 folder の file は確認作業のみ許可し、指示がない限り編集・移動・削除・生成してはいけない。
- `STATUS.md` は `C:\BOT\MLStock\STATUS.md` に置く。
- 変更時は `STATUS.md` を更新し、仕様変更・修正・運用変更・検証結果のいずれかを必ず記録する。
- `STATUS.md` の記述粒度は「何を、なぜ、何に影響するか」が追える最小単位とする。
- `STATUS.md` への追記は既存見出しにぶら下げ、日付付きで簡潔に追記する。
- 単発メモのために履歴系 `.md` を増やさず、まず `STATUS.md` へ集約する。
- `STATUS.md` に未確認事項を確定事項として書いてはいけない。
